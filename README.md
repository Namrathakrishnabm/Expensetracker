# Expense Tracker

A desktop-based Expense Tracker application built using Python, Tkinter, SQLite, and Matplotlib.

## Features

* Add Income and Expenses
* Category-based Expense Tracking
* Transaction History
* Date & Time Tracking
* SQLite Database Integration
* Automatic Balance Calculation
* Pie Chart Visualization
* CSV Export
* Delete Transactions
* Error Handling and Popups

## Technologies Used

* Python
* Tkinter
* SQLite
* Matplotlib
* CSV

## How to Run

1. Install Python
2. Install Matplotlib

```bash
python -m pip install matplotlib
```

3. Run the application

```bash
python expense_tracker_gui.py
```
